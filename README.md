# SimplePokedex

Simple pokedex showing you 100 pokemon

## Technology and open source libraries used
* Java
* Minimum SDK level 21
* Clean architecture
* Retrofit2
* Glide
* RxJava

## PokeApi
The pokeapi api was used to obtain the information of the pokemon
